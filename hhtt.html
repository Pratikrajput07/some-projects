Array

An array is one of the most fundamental and widely used data structures in computer science. It stores multiple values of the same data type under a single variable name in a contiguous block of memory. Each element is accessed using a unique numeric index, starting from 0. Arrays make it fast and easy to store, retrieve, and manipulate large amounts of related data — the backbone of many algorithms.

// 03  —  Definition

What is an Array?

An array is a data structure that stores many values of the same type in a single variable. Instead of creating many separate variables, an array keeps all values in continuous memory locations, placed side by side. Each value is called an element, and every element has an index starting from 0, which makes it easy to access any value by its position. The size of an array is fixed, so it cannot be changed after creation. Arrays are helpful for storing large sets of similar data like marks, prices, numbers, or names in an organized and efficient way.

arr = [10, 20, 30, 40]

Arrays make data handling easy, fast, and efficient.

// 04  —  Classification

Types of Array

TYPE 01
One-Dimensional Array (1D Array)
A one-dimensional array is the simplest form of array where all elements are stored in a single row or a single list. It uses only one index to access each element. This type of array is mostly used when data is arranged in a straight sequence, such as marks of students or prices of products.
marks = [85, 90, 78, 92]
TYPE 02
Two-Dimensional Array (2D Array)
A two-dimensional array stores data in rows and columns, similar to a table or matrix. It uses two indexes—row number and column number—to access each value. This type of array is helpful for representing structured data like grids, tables, or matrices used in mathematics.
matrix = [[1, 2], [3, 4]]
TYPE 03
Multi-Dimensional Array
A multi-dimensional array contains more than two dimensions. It is used for storing complex and layered structured data, such as 3D models, scientific datasets, or multi-level tables. These arrays become more powerful as dimensions increase, allowing deeper data representation.
arr = [[[1, 2], [3, 4]]]
// 05  —  Operations

Array Operations

OP_01
Traversing (Accessing Elements)
Traversing means visiting each element of the array one by one. It is used to display, check, or process every value stored in the array.
OP_02
Insertion (Adding Elements)
Insertion means adding a new element at a specific position. When inserting, existing elements may need to shift to make space.
OP_03
Deletion (Removing Elements)
Deletion removes an element from a specific position. After deletion, remaining elements shift to fill the empty space.
OP_04
Searching (Finding Elements)
Searching locates the position of a given element. Two types: Linear Search and Binary Search.
OP_05
Updating (Changing Value)
Updating means modifying or replacing the value of an existing element in the array at a specific index.
OP_06
Sorting (Arranging Elements)
Sorting arranges elements in ascending or descending order. Examples: Bubble Sort, Selection Sort, Insertion Sort.
OP_07
Merging (Combining Arrays)
Merging means combining two or more arrays into a single array.
OP_08
Splitting (Dividing Arrays)
Splitting divides one array into two or more smaller arrays based on a condition or index position.
// 06  —  Analysis

Advantages & Disadvantages

✦  Advantages
▶Arrays let you store many values in one variable.
▶Accessing any element with its index is very fast.
▶Elements are stored in continuous memory, making processing easier.
▶Using loops with arrays makes tasks like searching or printing simple.
▶Arrays help in performing operations like sorting and searching efficiently.
✕  Disadvantages
▶Array size is fixed; you cannot change it after creation.
▶If the array is too large, unused space wastes memory.
▶If the array is too small, it cannot store extra elements.
▶Arrays store only one datatype at a time.
▶Inserting or deleting elements in the middle is slow because shifting is needed.
